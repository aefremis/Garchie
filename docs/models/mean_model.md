# Mean Model (ARIMA) Documentation

## Overview
The `mean_model` class provides an automated pipeline for fitting an AutoRegressive Integrated Moving Average (ARIMA) model, specifically utilizing Seasonal ARIMA (SARIMAX) structures via `pmdarima` and `statsmodels`.

## Usage
The model accepts a pandas DataFrame containing `date` and `typical_price`.

```python
from garchie.models.mean_model import mean_model

# Initialize the ARIMA model
mm = mean_model(
    ts=ts_data, 
    train_size=0.8, 
    forecast_ahead=12, 
    stationarity=False, 
    diagnostics=True
)

# Run the optimization and forecasting pipeline
predictions = mm.design_mean_model()
```

## Key Components
- **Stationarity Testing:** Includes functionality to run a Dickey-Fuller test to check for stationarity and calculate the required differencing (`d` or `D` terms).
- **Seasonality Detection:** Automatically detects the dominant seasonal period (`m`) by analyzing the Autocorrelation Function (ACF) of the detrended time series.
- **Auto-ARIMA Optimization:** Uses `pmdarima.auto_arima` to execute a grid search over `p`, `d`, `q` (and their seasonal counterparts `P`, `D`, `Q`) to find the model with the lowest AIC.
- **Retraining:** The model splits the data for validation, but the final forecast is generated by feeding the previously discovered optimal `order` and `seasonal_order` into a fresh `statsmodels.tsa.statespace.sarimax.SARIMAX` model trained on the *entire* dataset.

## Returns
A standardized DataFrame containing future price predictions:
`['date', 'prediction', 'model_name', 'variable', 'lower_bound', 'upper_bound']`
